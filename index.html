ARDUINO

Skittles Sorter
A fun first prototype in the world of Arduino. Using a colour sensor, two servo motors and gravity, this machine sorts a bag of original Skittles into its respective flavours. Made for the OCD in us all.

Ardui-Bone
Took a crack at recreating my instrument with the Arduino; the trombone. This uses a sound (volume) sensor, gyroscope and an ultrasonic sensor to emulate playing the trombone through the Arduino's official tone library.

Habit-at Tracker
Took a stab at physical data visualization and PC-Arduino communication. This prototype tracks your habits by watering (or neglecting) your plants. Uses a mini water pump, relay and servo motor and communicates with the PC using a Processing App written in Java.

Mood Slippers
A take on smart home automation. Took the concept of Arduino-PC serial communication and made it wireless. Very simple wireless controller setup that cast music to your T.V, pours you a glass of wine and turns on the lights.

WEB

Trump Button
Fun little project that tasked us to explore the limits of what a digital button can do. You can make Trump do various things all within the button.

Ramen Rater
A collaborative effort to visualize ramen data compiled by the Ramen Rater. Uses the D3 JS library and has multiple interactive visual elements that update with each other.

Key Unlocker
A prototype that re-imagines the way you can unlock your phone using Android's physical sensors. It uses your phone's built in gyroscope to unlock the phone using a twist of the wrist. This app was written in Javascript using React Native.

Multiplayer Connect-4
Classic Connect-4 game using Sockets (NodeJS), HTML, CSS and Javascript. Allows players to create rooms, join friends' rooms or random open rooms. There is also the ability to change the colour theme, player name and room name.

OTHER

Peg Solitaire Solver
Interested in playing around with functional languages, this solver is written in Prolog and uses brute force with techniques to optimize solving speed. Solves the main board and other example layouts showing the time taken and individual moves.

No Fapp
Personal project still in the works, this tracks the progress of forgoing porn and masturbation using a simple timer algorithm. This also tracks the longest streaks, streak history and offers words of motivation to help curb this bad habit.
