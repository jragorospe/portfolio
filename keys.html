Key Unlocker

Tasked to invent a new way to unlock your phone using sensor inputs we created a motion key unlock. It was written in JavaScript and implement using React Native.

Github Repository

Context

The goal of this project was to design a unlock mechanism to replace the classic "slide-to-unlock" found on the early generation iPhones. The project imagines unlocking your phone using sensors and physical input. We have implemented a virtual key unlock that requires one to choose a key and complete a wrist twisting motion.

Rough Sketches

There were many sensors available to us via Android and iOS. Many of the ideas focused on the accelerometer which utilized gestures to unlock the phone in many themed ways.

Refined Sketches

We had a lot of fun designing this project. We were tasked to design an unlock using sensors on the phone. We had a lot of ideas for this but landed on the idea with the help of our classmates of unlocking your phone using the accelerometer in order to really focus on the sensor recognition. We wanted to make sure that the feel of unlocking the phone was precise and added our key designs to help prevent accidental unlocks and provide extra security. The design sketches reflected on the idea of unlocking using the correct key among multiple different key designs. Each key would use the same gesture to unlock but would either show a red screen with auditory feedback if the key chosen was incorrect or a green screen with auditory feedback that would eventually locked the phone if correct. Our algorithm checks for the position of the x-axis in order to check whether or not the phone is facing up or down. The audio will play and screen will change once the phone is flipped. The program detects the screen color and will play the correct audio accordingly. If green, the phone will wait for 3 seconds before unlocking your phone. This project was completed by Alexander Lam who wrote the logic and me who created all the assets and designed the interaction. It was implemented using React Native and coded in JavaScript.
