Ramen Rater

This was our main project for our data visualization class. We were tasked to find an interesting data set that we wanted to visualize using the D3.js library. We stumbled on this man's data set, where he has rated over 3500 different kinds of ramen for the last 18 years. We thought it best to use this one given its size and subject matter. Since it was our first time creating a data visualization we wanted to use a data set that we could fully encapsulate and one that didn't take itself so seriously. We ended up with an interactive dashboard where each section manipulates the others.

Interactions

1. Hover on map, tooltip with average rating and country name.
2. Single click on map, highlight matching bar on bar chart.
3. Double click on map, changes bar chart to average rating per type.
4. Hover on sunburst, tooltip with count and information.
5. Click on map, change country of data in sunburst.
